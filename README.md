# Kenya Macroeconomic Indicators Analysis
### Using Central Bank of Kenya (CBK) Data

## Project Overview

This project analyzes key macroeconomic indicators in Kenya using datasets published by the Central Bank of Kenya (CBK). The goal is to explore relationships between monetary policy, inflation dynamics, trade performance, and economic growth.

The analysis focuses on four pillars of the Kenyan economy:

- Monetary and Financial Statistics
- Inflation Trends
- National Accounts (GDP)
- External Trade and Balance of Payments

Using Python and R, the project performs exploratory data analysis, time series visualization, and macroeconomic interpretation of trends in the Kenyan economy.

This repository is designed to demonstrate data cleaning, exploratory analysis, economic reasoning, and reproducible research workflows.
